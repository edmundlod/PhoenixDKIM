About this source-code archive

All PhoenixDKIM source tarballs are signed with the release PGP key. The source repository on GitHub also carries signed release tags. Debian packages built by the release workflow are attached to each GitHub Release.

To verify a tarball, first import the release key: 

gpg --fetch-keys https://www.phoenixdkim.org/phoenixdkim-releases.asc

Then verify the signature: 

gpg --verify phoenixdkim-3.0.0-beta11.tar.gz.asc phoenixdkim-3.0.0-beta11.tar.gz

PhoenixDKIM 3.0.0 beta release

Beta releases are pre-release versions of PhoenixDKIM 3.0.0. They receive regular updates, including security fixes. The project author runs PhoenixDKIM in production on this basis; however, until a stable release is made, caution is advised for critical deployments. Only the most recent beta is maintained; all earlier betas are superseded.

  • PhoenixDKIM 3.0.0-beta11   (21 May 2026) Source tarball | PGP signature | GitHub Release (includes Debian packages) | Release notes

Note: beta1 through beta11 were released under the opendkim-ng name before the project was renamed to PhoenixDKIM on 21 May 2026.
