The PhoenixDKIM Home Page

Welcome the PhoenixDKIM project, and thank you for your interest.

What is PhoenixDKIM? It is a mail filter for DKIM verifying and signing, with a focus on security, correctness, and compatibility with current platforms. It aims (for now) to be a drop-in replacement for existing OpenDKIM deployments.

PhoenixDKIM was forked from OpenDKIM early 2026 as a personal project to bring the seemingly abandonned OpenDKIM up to date, so that I could keep using it, rather than switch to another DKIM solution.

As an Internet facing daemon, handling raw data directly from external networks, it is critical that it be kept up to date, and is as secure and correct as possible.

About this website

This website has information about the PhoenixDKIM source code distribution. Built from source code, Postfix can run on UNIX-like systems including Linux and BSD. It might run on more systems, but this has not been tested by the author.
