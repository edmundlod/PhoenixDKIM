About PhoenixDKIM

PhoenixDKIM is a maintained fork of OpenDKIM, a widely-deployed milter providing DKIM signing and verification for SMTP mail servers.

OpenDKIM is no longer actively maintained. PhoenixDKIM picks up where it left off: fixing bugs, updating defaults, and keeping the codebase current with modern mail infrastructure requirements.

PhoenixDKIM is compatible with any MTA that supports the milter protocol, including Postfix and Sendmail.
